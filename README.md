# Unity Font Tool v1.0.0
 **Unity Font Tool v1.0.0** は、UnityのTextMeshPro Font Asset Creator用に必要な文字だけを簡単に抽出・整理できるWindows向け支援ツールです。TXT・DOCX・フォルダー一括読み込み・テキスト貼り付け入力に対応し、使用文字や漢字のみを抽出できます。重複文字の削除、空白・改行の除去、出現順維持、UTF-8形式での保存にも対応。実際に制作中の東方Project二次創作ゲーム『東方恋華録』の開発効率化を目的として開発しました。Unityでのフォントアセット作成を、より手軽で快適にサポートします。

> Unity TextMeshPro Font Asset Creator向け文字抽出支援ツール

Unity Font Toolは、UnityのTextMeshPro Font Asset Creatorで使用する文字セットを効率よく作成するためのWindows向け支援ツールです。

ゲーム制作中のシナリオやテキストから**実際に使用する文字だけ**を抽出し、フォントアセットの作成をより簡単・効率的に行えます。

---

## ✨ 主な機能

- 📄 TXTファイル読み込み
- 📝 DOCXファイル読み込み
- 📂 フォルダー内ファイル一括読み込み
- 📋 テキスト貼り付け入力対応
- 🔤 全使用文字抽出
- 漢字のみ抽出
- 重複文字自動削除
- 空白・改行自動除去
- 出現順を維持したまま抽出
- UTF-8形式で保存
- Windows向けGUIアプリ

---

## 🚀 使い方

### ① ファイルを読み込む

以下の方法に対応しています。

- TXT
- DOCX
- フォルダー一括
- テキスト貼り付け

---

### ② 抽出モードを選択

- 全使用文字
- 漢字のみ

---

### ③ 抽出

不要な

- 重複文字
- 空白
- 改行

を自動で除去します。

---

### ④ 保存

UTF-8形式のテキストとして保存します。

---

### ⑤ Unityへ

保存した文字列を

**TextMeshPro Font Asset Creator**

の

**Custom Characters**

へ貼り付けるだけです。
